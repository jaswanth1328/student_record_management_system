Overview

The Student Record Management System is a web-based application designed to manage student data efficiently. It provides an intuitive interface for adding, updating, viewing, and deleting student records. Built with HTML, CSS, and JavaScript, it features smooth animations, responsive layouts, and interactive components for a seamless experience.

Key Features

Add/Update Student Records: Easily input and modify student details. Responsive Tables: View all student records in a structured table. Interactive Cards: Quick actions through animated card UI. Modal Popups: Edit or confirm actions with smooth modal animations. Search and Sort: Filter and organize student records efficiently. Modern UI: Gradient backgrounds, transitions, and hover effects enhance usability.

How to Use

Open index.html in your preferred browser. Add a new student by filling in the form fields and clicking submit. View students in the table below the form. Edit or Delete student records via action buttons. Interact with cards and modals for a more visual experience.

Why This Program

Helps manage student records efficiently. Provides an intuitive, user-friendly interface. Reduces manual errors in record keeping. Demonstrates front-end development skills (HTML, CSS, JS). Useful for schools, colleges, or small educational institutes
